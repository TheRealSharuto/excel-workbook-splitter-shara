Excel Data Extractor

  1. Choose your excel file
  2. Enter the column name of the value you will be filtering by
  3. If you only want one workbook generated for one value, enter the value. If you want workbooks generated for each unique value in the column, enter 0.
  4. Enter the name of the new workbook you are generating
  5. Click extract, the file will be downloaded to Downloads (or wherever your default downloads location is)

Note: If you are extracting by a column with no value, leave Enter column value: blank.
