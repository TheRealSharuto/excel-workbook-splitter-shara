Excel Splitter

  1. Choose your excel file
  2. Enter the amount of rows you want in each generated workbook (i.e. 500)
  3. Enter the name of the new workbooks you are generating. Each workbook will have the name and number (For example, the name is Interos and 5 workbooks get generated. The names will be Interos1.xlsx, Interos2.xlsx, Interos3.xlsx etc.)
  4. Click split, a zip file (with the name you chose) with all of the workbooks will be downloaded to Downloads (or wherever your default downloads location is)
